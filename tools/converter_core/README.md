# Vendored converter utilities

This directory contains the non-GUI conversion utilities extracted from `ReverseTelnet/ps4_ps5_aio_cheat_converter`, including project-maintained fixes used by this catalog.

Only the JSON/SHN/MC4 conversion modules needed by this catalog are included. GUI entry points, sample files, and script output folders are intentionally omitted.
